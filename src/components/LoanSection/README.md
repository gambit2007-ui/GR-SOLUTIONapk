# LoanSection

Este modulo concentra a gestao de contratos, incluindo:

- criacao de contratos
- edicao de contratos
- listagem e filtros
- baixa de parcelas
- renovacao de juros
- quitacao antecipada
- integracao com WhatsApp

## Observacoes

O componente foi mantido com a mesma API publica para evitar regressoes.

Futuras refatoracoes devem separar:

- formulario de contrato
- card/listagem
- modais de pagamento
- renovacao de juros
- regras financeiras
